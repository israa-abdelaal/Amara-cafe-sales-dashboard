# ☕ Amara Cafe Sales Dashboard

An interactive Power BI dashboard built to analyze and track 
the sales performance of Amara Cafe across multiple branches.

## 📊 Dashboard Overview

| Metric | Value |
|--------|-------|
| Total Sales | 714K |
| Total Cheque | 560K |
| Total Cash | 154K |
| Total Transactions | 279 |
| Net Quantity | 5K |
| Cancelled Quantity | 57 |

## 🔍 Key Features

- **Monthly Sales Trend** — Track sales performance over time
- **Payment Method Analysis** — Cash vs Cheque breakdown
- **Location Filter** — Dynamic filtering by branch
- **Top 5 Products** — Best selling items by revenue
- **Sales by Location** — Compare branches performance

## 🏪 Branches Analyzed
- Mecca Mall
- Al Shawqiya
- Wadi Al-Ahd

## 🛠️ Tools Used
- Microsoft Power BI
- DAX for calculations
- Power Query for data transformation

## 📈 Insights
- Sales grew by **1.28%** compared to previous period
- **Wafer Frappe with Chocolate** is the top selling product (8.89K)
- **Al Shawqiya** branch leads in total sales (20K)
